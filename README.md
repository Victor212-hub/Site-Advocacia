# Advocacia Anderson Luciano — Site Institucional

Site institucional do escritório **Dr. Anderson Luciano Advocacia & Consultoria Jurídica**, desenvolvido para apresentar os serviços, facilitar o contato e orientar clientes sobre como chegar ao escritório.

---

## Estrutura do projeto

```
advocacia-anderson-luciano/
├── index.html
├── style.css
└── img/
    ├── logo_advocia.png
    └── foto_frente_advocacia.jpeg
```

---

## Seções do site

| Seção | Descrição |
|---|---|
| **Header / Nav** | Navegação fixa com logo, links e botão de WhatsApp |
| **Hero** | Apresentação do escritório com chamada para contato |
| **Áreas de Atuação** | Cards com as áreas do direito atendidas |
| **Previdenciário** | Destaque para as ações previdenciárias |
| **Fachada** | Foto da frente do escritório para facilitar a chegada |
| **Localização** | Endereço completo + mapa do Google Maps incorporado |
| **Rodapé** | Contato, navegação e copyright |

---

## Tecnologias utilizadas

- **HTML5** semântico (`header`, `main`, `section`, `address`, `footer`, `figure`)
- **CSS3** puro — sem frameworks, sem dependências
- **Google Fonts** — Playfair Display + Lato
- **Google Maps Embed** — mapa incorporado via iframe
- **WhatsApp API** — link direto via `wa.me`

---

## Responsividade

O layout se adapta a três tamanhos de tela:

- **Desktop** — navegação horizontal completa
- **Tablet** (até 900px) — mapa e endereço em coluna única
- **Mobile** (até 680px) — menu hamburguer, cards em coluna única

O menu hamburguer no mobile funciona **sem JavaScript**, usando apenas CSS com o truque de `checkbox` + `label`.
